# Vehicle-Detection

A Deep-Learning Approach for Detection and Classification of Vehicles for Traffic Management.

Developed by Abubakar Isa.

## Architecture

This project incorporates a real-time web application (Flask) supporting side-by-side inference generation between YOLOv11 and Faster R-CNN for localized traffic detection.
